# Advanced Search & Data Table - Candidate Management

This is a submission-ready frontend project implementing the "Advanced Search & Data Table for Candidate Management" assignment.

## Features
- Large mock dataset (10k records)
- Advanced filters (name, company, stage, salary range)
- Virtualized data table (react-window) for performance
- Sorting, inline edits, bulk selection/actions
- Candidate detail modal
- View More / View Less for lists
- Tailwind CSS for styling

## Setup
1. Install dependencies:
   npm install

2. Run dev server:
   npm run dev

3. Build:
   npm run build

## Notes
This is a front-end demo; in a real submission, connect API endpoints for persistence.
